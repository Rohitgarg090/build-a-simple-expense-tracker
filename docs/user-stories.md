## User Stories

### Authentication
- **US-01** As a new user, I want to register with my email and password so that I can create a personal account.
- **US-02** As a returning user, I want to log in with my credentials so that I can access my expense data.
- **US-03** As a logged-in user, I want to be automatically redirected to the dashboard so that I don't have to navigate manually after login.
- **US-04** As a user, I want to log out so that my account is secure on shared devices.

### Expense Management
- **US-05** As a user, I want to add a new expense with an amount, category, date, and title so that I can record my spending.
- **US-06** As a user, I want to add optional notes to an expense so that I can remember context (e.g., merchant name).
- **US-07** As a user, I want to view a list of all my expenses so that I can see my spending history.
- **US-08** As a user, I want to click on an expense to view its full details so that I can review the information.
- **US-09** As a user, I want to edit an existing expense so that I can correct mistakes.
- **US-10** As a user, I want to delete an expense so that I can remove incorrect or duplicate entries.

### Categories
- **US-11** As a user, I want to see default categories available on first use so that I can start logging expenses immediately.
- **US-12** As a user, I want to create a custom category with a name and color so that I can organize expenses my way.
- **US-13** As a user, I want to rename a category so that I can keep my labels meaningful.
- **US-14** As a user, I want to delete a category I no longer need so that my category list stays clean.

### Dashboard
- **US-15** As a user, I want to see my total spending for the current month on the dashboard so that I know where I stand.
- **US-16** As a user, I want to see a category breakdown chart so that I can understand where my money is going.
- **US-17** As a user, I want to see a daily spending trend over the last 30 days so that I can spot patterns.
- **US-18** As a user, I want to see my 5 most recent expenses on the dashboard so that I have quick visibility into latest activity.

### Filtering & Search
- **US-19** As a user, I want to filter my expense list by category so that I can focus on a specific type of spending.
- **US-20** As a user, I want to filter by a date range so that I can review spending for a specific period.
- **US-21** As a user, I want to search expenses by title or notes so that I can quickly find a specific record.
- **US-22** As a user, I want to sort expenses by date or amount so that I can view them in a meaningful order.

### Settings
- **US-23** As a user, I want to update my display name so that my profile reflects my preferred name.
- **US-24** As a user, I want to select a preferred currency so that amounts are displayed in my local format.
- **US-25** As a user, I want to delete my account and all data so that I can leave the platform cleanly.
